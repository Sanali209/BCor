3.1 No REST/HTTP API

BCor не использует FastAPI или REST endpoints. В проекте нет FastAPI-приложений. Коммуникация — через асинхронную шину событий (MessageBus/bubus).

Единственная асинхронная HTTP-точка — uvicorn как зависимость для TaskIQ dashboard.

3.3 Command Dispatch Flow

Все entry points следуют одному паттерну:

1. System.from_manifest() — bootstrap из app.toml
2. system.start() — инициализация DI контейнера (dishka)
3. system.container() as scope — получение request scope
4. scope.get(MessageBus) — получение шины событий
5. bus.dispatch(Command) — отправка команды
6. Обработчик выполняется → возвращает BusinessResult
7. system.stop() — корректное завершение
