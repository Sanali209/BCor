# ADR 0001: Использование паттерна "Душитель" для портирования Image Analyze на BCor

## Статус
Принято (Accepted)

## Контекст
Приложение Image Analyze является монолитным десктопным приложением на PySide6. Нам необходимо перенести его на модульный фреймворк BCor, сохранив работоспособность на каждом этапе.

## Решение
Мы используем паттерн **"Душитель" (Strangler Fig)**. 
1.  **Фаза 1**: Обертывание всего легаси-приложения в модуль BCor. Точка входа приложения заменяется на BCor `System`, которая через `MessageBus` запускает старый GUI.
2.  **Фаза 2**: Постепенное выделение бизнес-логики в чистый Domain Core (Hexagonal Architecture) с использованием TDD.
3.  **Фаза 3**: Замена вызовов в GUI на команды BCor, превращая GUI в "тонкий" адаптер.

## Последствия
- **Положительные**: Приложение остается в рабочем состоянии на протяжении всего процесса миграции. Риски регрессии минимизированы.
- **Отрицательные**: Временное усложнение структуры (обертка + легаси код). Необходимость поддержки двух систем импортов на переходном этапе.
