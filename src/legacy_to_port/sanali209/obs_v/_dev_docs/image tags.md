
# Tag Conventions

Tags are hierarchical, with each level separated by a `/`.  
For example: `tag1/tag2/tag3` indicates that `tag1` is the parent of `tag2`, and `tag2` is the parent of `tag3`.

## Tag Prefixes

- Tags created using the class indexer `"from name"` start with `from_name/`.
- Tags created using the class indexer `"deepdanboru"` start with `auto/deepdanboru/`.
- Tags imported from Boru data start with `imported/boru/`.
- Tags describing image copyrights start with `copyright/`.
- Tags describing the image author start with `author/`.
- Tags describing characters in the image start with `character/`.
- Tags describing character conditions in the image start with `character condition/`.
- Tags describing actions in the image start with `action/`.
