# Vector DB Module

Этот модуль отвечает за управление векторными базами данных (эмбеддингами) и выполнение поиска по сходству (ближайших соседей) в SLM. Он использует библиотеку `Annoy` для создания индексов в памяти.

## Структура модуля

*   **`vector_db.py`**: Содержит [основные классы](vector_db.md) (`VectorDB`, `VectorDBPreferences`, `SearchScope`, `SearchScopeList`) для регистрации конфигураций индексов, создания индексов Annoy в памяти и выполнения поиска.
*   **`vector_db_module.py`**: Вероятно, содержит класс модуля для интеграции функциональности векторной БД с основной архитектурой приложения SLM (`SLM.appGlue.core`). *(Требуется дальнейшее изучение)*.

## Основные возможности

*   Регистрация различных конфигураций векторных индексов (размерность, метрика расстояния, функция векторизации).
*   Создание индексов Annoy в памяти на основе списка элементов.
*   Поиск ближайших соседей по элементу или по вектору.
*   Поиск потенциальных дубликатов или близких элементов внутри набора данных.
*   Использование различных метрик расстояния Annoy (`angular`, `euclidean`, `manhattan`, `hamming`, `dot`).
