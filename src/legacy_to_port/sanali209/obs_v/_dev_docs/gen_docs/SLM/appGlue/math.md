# Math Utilities (`math.py`)

Этот файл содержит простые вспомогательные математические функции общего назначения.

## Функции

*   **`Remap(value, min1, max1, min2, max2)`**
    *   **Назначение:** Линейно преобразует (перемасштабирует) значение `value` из одного диапазона (`min1` до `max1`) в другой диапазон (`min2` до `max2`).
    *   **Параметры:**
        *   `value`: Исходное значение для преобразования.
        *   `min1`: Нижняя граница исходного диапазона.
        *   `max1`: Верхняя граница исходного диапазона.
        *   `min2`: Нижняя граница целевого диапазона.
        *   `max2`: Верхняя граница целевого диапазона.
    *   **Возвращает:** Значение, перемасштабированное в целевой диапазон.
    *   **Пример:** `Remap(25, 0, 100, 0, 1)` вернет `0.25`.

*   **`clip(value, lower, upper)`**
    *   **Назначение:** Ограничивает значение `value` так, чтобы оно находилось в пределах заданного диапазона [`lower`, `upper`].
    *   **Параметры:**
        *   `value`: Значение для ограничения.
        *   `lower`: Нижняя граница диапазона.
        *   `upper`: Верхняя граница диапазона.
    *   **Возвращает:**
        *   `lower`, если `value` меньше `lower`.
        *   `upper`, если `value` больше `upper`.
        *   `value`, если оно находится между `lower` и `upper` (включительно).
