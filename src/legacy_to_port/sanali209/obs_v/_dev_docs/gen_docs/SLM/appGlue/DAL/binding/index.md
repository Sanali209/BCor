# Data Binding (`binding/`)

Этот подмодуль `DAL` содержит реализацию системы связывания данных (Data Binding).

## Основные компоненты

*   [`bind.py`](bind.md): Определяет классы `BindingProperty`, `BindingObserver`, `PropBinding`, `PropUser`, `PropInfo` и `BindFactoryBase` для создания и управления связями между свойствами объектов, включая поддержку конвертеров и двустороннего связывания.

Система связывания данных позволяет автоматически синхронизировать значения между различными частями приложения, например, между моделью данных (или ViewModel) и элементами пользовательского интерфейса.
