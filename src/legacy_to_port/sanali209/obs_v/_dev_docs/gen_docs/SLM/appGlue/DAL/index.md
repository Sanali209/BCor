# Data Access Layer (`DAL/`)

Подмодуль `DAL` (Data Access Layer) в `appGlue` предоставляет абстракции и конкретные реализации для работы с данными, их хранением, преобразованием и представлением. Он включает механизмы для описания схем данных, работы с коллекциями (списками) и древовидными структурами, а также интеграцию с внешними источниками, такими как MongoDB.

## Основные компоненты и файлы

*   [`DAL.py`](DAL.md): Определяет базовые концепции DAL, включая схемы данных (`GlueObjectDataScheme`), адаптеры (`DataAdapter`), мапперы (`DataSchemeMapper`), абстрактный `DataProvider`, конвертеры (`GlueDataConverter`) и селектор шаблонов (`AdapterTemplateSelector`).
*   [`datalist2.py`](datalist2.md): Реализует паттерн Наблюдатель для моделей данных (`AbstractDataModel`, `DataListModel`, `MongoDataModel`), логику запросов (`AbstractDataQuery`, `DataQuery`, `MongoDataQuery`) и представление данных с пагинацией и фильтрацией (`DataViewCursor`).
*   [`treedata.py`](treedata.md): Предоставляет `TreeDataView` для создания фильтруемых и наблюдаемых представлений над древовидными структурами данных, хранящимися в статическом словаре.
