# Application Glue Module (`appGlue`)

Модуль `appGlue` предоставляет набор основных сервисов, утилит и паттернов, которые служат "клеем", связывающим различные части приложения SLM. Он включает в себя фундаментальные компоненты для управления жизненным циклом приложения, конфигурацией, ресурсами, событиями, фоновыми задачами и другими общими механизмами.

## Основные компоненты и файлы

*   [`core.py`](core.md): Ядро модуля, определяющее базовые классы для ресурсов (`Resource`), сервисов (`Service`), модулей (`Module`), системы конфигурации (`BaseConfig`, `ConfigSection`), управления бэкендами (`ServiceBackend`, `BackendProvider`), центрального распределителя (`Allocator`) и основного класса приложения (`GlueApp`).
*   [`hashingService.py`](hashingService.md): Сервис для кэширования результатов функций на диске с использованием `diskcache` и декоратора `@HashingService.hashable`.
*   [`helpers.py`](helpers.md): Набор вспомогательных функций для работы с изображениями (Base64, сжатие, миниатюры, MD5), а также специфичная функция для получения URL ngrok.
*   [`math.py`](math.md): Простые математические утилиты (`Remap`, `clip`).
*   [`progress_visualize.py`](progress_visualize.md): Система для отслеживания и отображения прогресса длительных задач (`ProgressManager`, `ProgressVisualizer`).
*   [`selectionManager.py`](selectionManager.md): Система для управления выбором элементов (`SelectionManager`, `SelectionManagerUser`).
*   [`timertreaded.py`](timertreaded.md): Система для управления таймерами, работающими в отдельном потоке (`TimerManager`, `Timer`, `TObserver`).
*   [`undo_system.py`](undo_system.md): Простая реализация механизма Отмены/Повтора (`UndoSystem`, `UndoItem`).
*   `core_docks.md`: Существующий файл Markdown, возможно, содержащий дополнительную документацию или заметки по ядру. *(Не проанализирован)*
