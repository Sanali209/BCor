# Metadata Module

Этот модуль отвечает за извлечение, обработку и управление метаданными файлов в рамках SLM. Он может использовать как встроенные библиотеки Python (например, `PIL`), так и внешние инструменты (например, `exiftool`).

## Структура модуля

*   **`functions.py`**: Содержит набор [функций для работы с метаданными](functions.md) изображений (IPTC, XMP) с использованием `iptcinfo3` и `PIL`.
*   **`MDManager/`**: Поддиректория, вероятно, содержащая основной класс или логику для управления метаданными. *(Требуется дальнейшее изучение содержимого)*.
*   **`exiftool.exe`** (находится в `Python/SLM/`): Внешний инструмент, который может использоваться этим модулем для извлечения широкого спектра метаданных из различных типов файлов.

*(Общее описание того, как эти компоненты взаимодействуют, будет добавлено после анализа `MDManager/`)*.

## Основные возможности (Предполагаемые)

*   Извлечение различных типов метаданных (EXIF, IPTC, XMP, специфичные для формата).
*   Чтение и запись метаданных.
*   Нормализация и структурирование извлеченных метаданных.
*   Интеграция с другими модулями SLM (например, `files_db` для хранения, `Graph` для представления связей).
