# Tile Map System (`tileMap.py`)

Этот модуль реализует систему тайловой карты с оптимизированным рендерингом для 2D игр.

## Основные классы

### `TileMap`
```python
class TileMap(EnvironmentEntity):
    def __init__(self):
        self.tile_size = (16, 16)
        self.map_size = (640, 480)
        # ... другие поля ...
```

#### Атрибуты:
- `tile_size`: Размер тайла (по умолчанию 16x16)
- `map_size`: Размер карты в тайлах
- `tiles`: Двумерный массив индексов тайлов
- `tiles_set`: Набор доступных тайлов
- `sprite_groups`: Группы спрайтов для разных типов объектов

#### Группы спрайтов:
```python
self.sprite_groups = {
    "tiles": pygame.sprite.Group(),
    "player": pygame.sprite.Group(),
    "npc": pygame.sprite.Group(),
    "obstacle": pygame.sprite.Group(),
}
```

#### Методы:
1. **`addTile(tile)`**: Добавляет новый тайл в набор
2. **`GenerateTiles(width, height)`**: Создает случайную карту заданного размера

### `Tile`
```python
class Tile:
    def __init__(self):
        self.tile_sprite = Sprite()
        self.obstacle = False
```

#### Атрибуты:
- `tile_sprite`: Спрайт тайла (Pygame Sprite)
- `obstacle`: Флаг является ли тайл препятствием

#### Методы:
1. **`load_tile(tile_path, size, x, y)`**:
   - Загружает изображение тайла из спрайтового листа
   - Масштабирует до нужного размера
   - Устанавливает прозрачность

2. **`get_copy()`**:
   - Создает копию тайла
   - Используется для оптимизации памяти

### `PyGameTileMapRenderer`
```python
class PyGameTileMapRenderer(EntityController):
    def __init__(self):
        super().__init__()
```

#### Методы:
1. **`render()`**:
   - Обновляет видимую часть карты
   - Отрисовывает все группы спрайтов

2. **`update_tile_map(tile_map, camera)`**:
   - Определяет видимую область карты
   - Оптимизирует отрисовку видимых тайлов
   - Учитывает положение камеры

## Оптимизации рендеринга

### 1. Отрисовка видимой области
```python
start_x = math.ceil(camera_position[0] // tile_size[0])
start_y = math.ceil(camera_position[1] // tile_size[1])
end_x = start_x + (camera_position[0] + camera.view_size[0]) // tile_size[0]
end_y = start_y + (camera_position[1] + camera.view_size[1]) // tile_size[1]
```
- Вычисление границ видимой области
- Отрисовка только видимых тайлов
- Учет размера тайлов и позиции камеры

### 2. Управление памятью
- Переиспользование спрайтов тайлов
- Очистка группы спрайтов перед обновлением
- Эффективная работа с текстурами

### 3. Конвертация изображений
```python
self.tile_sprite.image.convert()
self.tile_sprite.image.set_colorkey((254, 254, 254))
```
- Преобразование формата для быстрого рендеринга
- Установка прозрачного цвета

## Использование

### Создание карты
```python
# Инициализация карты
tile_map = TileMap()

# Добавление тайлов
grass_tile = Tile()
grass_tile.load_tile('GRASS+.png', (16, 16), 0, 0)
tile_map.addTile(grass_tile)

# Генерация карты
tile_map.GenerateTiles(40, 30)  # 40x30 тайлов
```

### Интеграция с окружением
```python
# Добавление в игровую среду
environment.AddChild(tile_map)

# Добавление объектов на карту
player.AddToGroup(tile_map.sprite_groups['player'])
obstacle.AddToGroup(tile_map.sprite_groups['obstacle'])
```

## Особенности реализации

1. **Модульность:**
   - Разделение логики тайлов и рендеринга
   - Интеграция через систему EntityController
   - Поддержка различных типов спрайтов

2. **Производительность:**
   - Отрисовка только видимой области
   - Переиспользование спрайтов
   - Оптимизированная работа с текстурами

3. **Расширяемость:**
   - Поддержка различных типов тайлов
   - Гибкая система групп спрайтов
   - Возможность добавления новых функций

## Ограничения

1. **Функциональные:**
   - Только прямоугольные тайлы
   - Отсутствие слоев

   - Зависимость от Pygame
   - Отсутствие поддержки анимированных тайлов
