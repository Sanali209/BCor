# Cartographer Application (`cartographer/`)

Приложение для создания и редактирования тайловых карт, демонстрирующее возможности системы `botbox`.

## Структура каталога

```
cartographer/
├── __init__.py          # Инициализатор пакета
├── cart new.py          # Новая версия картографа
├── carto.py            # Основная версия картографа
├── init.png           # Начальное изображение
├── map.png            # Файл карты
├── result.jpg         # Результат работы
├── [Originals]/       # Оригинальные ресурсы
└── scr/               # Исходники и ресурсы
```

## Основные компоненты

### 1. Редактор карт (`carto.py`)
- Основной интерфейс для создания карт
- Работа с тайлами и слоями
- Сохранение/загрузка карт

### 2. Обновленный редактор (`cart new.py`)
- Улучшенная версия редактора
- Дополнительные инструменты
- Расширенные возможности

## Функциональность

1. **Создание карт:**
   - Размещение тайлов
   - Выбор из палитры
   - Редактирование слоев

2. **Редактирование:**
   - Инструменты рисования
   - Заливка областей
   - Работа с выделенными участками

3. **Управление файлами:**
   - Сохранение проектов
   - Экспорт карт
   - Импорт ресурсов

## Использование

### Базовое использование
```python
from SLM.botbox.app.cartographer.carto import MapEditor

# Создание редактора
editor = MapEditor()

# Загрузка карты
editor.load_map("map.png")

# Запуск интерфейса
editor.run()
```

### Расширенные возможности
```python
from SLM.botbox.app.cartographer.cart_new import AdvancedMapEditor

# Создание расширенного редактора
editor = AdvancedMapEditor()

# Настройка параметров
editor.set_tile_size(32, 32)
editor.enable_grid(True)

# Запуск с дополнительными опциями
editor.run(fullscreen=True)
```

## Ресурсы

1. **Изображения:**
   - `init.png`: Начальный шаблон
   - `map.png`: Пример карты
   - `result.jpg`: Пример результата

2. **Директории:**
   - `[Originals]/`: Исходные файлы ресурсов
   - `scr/`: Дополнительные ресурсы

## Интеграция с botbox

1. **Использование компонентов:**
   - Тайловая система (`TileMap`)
   - Рендеринг (`PyGameTileMapRenderer`)
   - Система сущностей (`EnvironmentEntity`)

2. **Расширения:**
   - Пользовательские тайлы
   - Новые инструменты
   - Дополнительные форматы

## Инструменты редактирования

1. **Базовые:**
   - Кисть (размещение тайлов)
   - Ластик (удаление тайлов)
   - Заливка (заполнение области)

2. **Продвинутые:**
   - Выделение области
   - Копирование/вставка
   - Работа со слоями

3. **Утилиты:**
   - Сетка
   - Линейки
   - Направляющие

## Горячие клавиши

- `Ctrl+S`: Сохранить карту
- `Ctrl+O`: Открыть карту
- `Ctrl+Z`: Отменить действие
- `Space`: Переключить сетку
- `1-9`: Выбор инструментов
- `Tab`: Переключение палитры
