# Example Applications (`app/`)

Этот каталог содержит примеры приложений, демонстрирующие использование функционала `botbox`.

## Структура

```
app/
├── __init__.py
├── cartographer/      # Приложение для работы с картами
└── py rpg/           # Пример RPG игры
```

## Приложения

### Cartographer

Приложение для создания и редактирования тайловых карт. Использует функционал:
- Тайловой системы (`TileMap`)
- Рендеринга (`PyGameTileMapRenderer`)
- Управления камерой

### Python RPG

Демонстрационная RPG игра, показывающая использование:
- Системы сущностей (`Actor`, `Player`, `Obstacle`)
- Деревьев поведения (`behTree`)
- Тайловых карт (`TileMap`)
- Системы анимации (`Animation`)

## Использование примеров

### Запуск приложений

```python
# Запуск картографа
from SLM.botbox.app.cartographer import main
main()

# Запуск RPG
from SLM.botbox.app.py_rpg import main
main()
```
