# Python RPG Demo (`py rpg/`)

Демонстрационная RPG игра, показывающая возможности фреймворка `botbox` для создания 2D игр.

## Структура проекта

```
py rpg/
├── __init__.py
├── animation.py               # Система анимации
├── behavior_funct.py         # Функции поведения NPC
├── bot_actions.py            # Действия ботов
├── bot_test.py              # Тесты ботов
├── helper.py                 # Вспомогательные функции
├── map_traker.py            # Отслеживание карты
├── mtraker.py               # Дополнительный трекер
├── rpg base.py              # Основной файл игры
├── rpg_ent.py               # Сущности
├── temp.py                  # Временные функции
├── tileMap.py               # Система тайлов
└── resources/               # Ресурсы игры
    ├── Bat_Sprite_Sheet.png    # Спрайты летучей мыши
    ├── GRASS+.png             # Текстуры травы
    ├── rock_obstacle.png      # Текстуры камней
    ├── SpriteSheet.png        # Общий набор спрайтов
    └── test_vector.png        # Тестовое изображение
```

## Основные компоненты

### 1. Игровая механика
- **`rpg base.py`**: Основной игровой цикл и логика
- **`rpg_ent.py`**: Определения игровых сущностей
- **`tileMap.py`**: Система тайловых карт

### 2. Поведение ИИ
- **`behavior_funct.py`**: Функции поведения NPC
- **`bot_actions.py`**: Действия ботов
- **`bot_test.py`**: Тестирование поведения

### 3. Вспомогательные системы
- **`animation.py`**: Система анимации
- **`helper.py`**: Утилиты
- **`map_traker.py`/`mtraker.py`**: Отслеживание состояния карты

## Игровые механики

1. **Движение персонажа:**
   - Управление с клавиатуры
   - Коллизии с препятствиями
   - Плавная анимация

2. **NPC и враги:**
   - ИИ на основе деревьев поведения
   - Патрулирование территории
   - Преследование игрока

3. **Окружение:**
   - Тайловая карта
   - Препятствия
   - Интерактивные объекты

## Разработка

### Добавление нового NPC
1. Создать класс, наследующий `Actor`
2. Определить поведение через BehaviorTree
3. Добавить спрайты и анимации
4. Зарегистрировать в игровом мире

### Создание новых действий
1. Определить действие в `bot_actions.py`
2. Создать узел дерева поведения
3. Интегрировать в существующие деревья

### Расширение карты
1. Добавить новые тайлы
2. Обновить карту в `tileMap.py`
3. Настроить коллизии
4. Добавить интерактивные объекты
