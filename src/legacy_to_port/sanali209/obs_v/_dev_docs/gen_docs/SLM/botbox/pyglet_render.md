# Pyglet Renderer (`pyglet_render.py`)

Этот модуль реализует простой интерфейс для рендеринга с использованием библиотеки Pyglet.

## Класс `PyGletRender`

```python
class PyGletRender:
    def __init__(self):
        self.game_screen_size = (800, 600)
        self.window = pyglet.window.Window(width=self.game_screen_size[0], 
                                         height=self.game_screen_size[1])
```

### Назначение

Предоставляет базовую обертку над Pyglet для создания окна и рендеринга графики.

### Атрибуты

- `game_screen_size`: Кортеж (width, height) с размерами окна (по умолчанию 800x600)
- `window`: Экземпляр `pyglet.window.Window` для отображения графики

### Зависимости

- `pyglet`: Основная библиотека для создания окна и рендеринга

### Особенности

1. **Минимализм:**
   - Базовая инициализация окна
   - Стандартные размеры по умолчанию
   - Нет дополнительной настройки

2. **Расширяемость:**
   - Может быть расширен для поддержки:
     - Обработки событий
     - Отрисовки спрайтов
     - Работы с текстом
     - Поддержки анимации

## Использование

```python
# Создание рендерера
renderer = PyGletRender()

# Изменение размера окна (если нужно)
renderer.game_screen_size = (1024, 768)
```

## Рекомендации по использованию

1. **Базовое использование:**
   ```python
   # Создание рендерера
   renderer = PyGletRender()
   
   # Использование окна Pyglet напрямую
   @renderer.window.event
   def on_draw():
       renderer.window.clear()
       # Дополнительная логика отрисовки
   ```

2. **Расширение функциональности:**
   ```python
   class ExtendedPyGletRender(PyGletRender):
       def __init__(self):
           super().__init__()
           self.sprites = []
           
       def add_sprite(self, sprite):
           self.sprites.append(sprite)
           
       def render(self):
           self.window.clear()
           for sprite in self.sprites:
               sprite.draw()
   ```
