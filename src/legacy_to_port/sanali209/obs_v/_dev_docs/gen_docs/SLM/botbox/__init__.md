# Package Initializer (`__init__.py`)

Этот файл `__init__.py` находится в директории `botbox` и является пустым. 

## Назначение

Несмотря на отсутствие кода, файл необходим для:
1.  Обозначения директории `botbox` как Python-пакета.
2.  Возможности импорта модулей из этого пакета, например:
    ```python
    from SLM.botbox import animation
    from SLM.botbox.rpg_ent import SomeClass
    ```

В будущем сюда можно добавить инициализационный код, импорты или определения, общие для всего пакета.
