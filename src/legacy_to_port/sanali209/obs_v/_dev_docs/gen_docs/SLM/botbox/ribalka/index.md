# Fishing Simulator (`ribalka/`)

Модуль реализует простой симулятор рыбалки как демонстрацию возможностей фреймворка `botbox`.

## Структура каталога

```
ribalka/
├── __init__.py       # Инициализатор пакета
├── riba.py          # Основной файл симулятора
├── poplavok.png     # Спрайт поплавка
└── Screenshot_3.png # Скриншот игры
```

## Компоненты

### 1. Основной модуль (`riba.py`)
- Игровая логика
- Обработка ввода
- Симуляция физики поплавка и рыбы

### 2. Ресурсы
- `poplavok.png`: Изображение поплавка
- `Screenshot_3.png`: Демонстрационный скриншот

## Геймплей

1. **Механика рыбалки:**
   - Забрасывание удочки
   - Наблюдение за поплавком
   - Подсечка при поклёвке
   - Вываживание рыбы

2. **Симуляция:**
   - Поведение рыбы
   - Физика поплавка
   - Влияние течения
   - Случайные события

3. **Взаимодействие:**
   - Управление удочкой
   - Выбор места заброса
   - Настройка снастей

## Конфигурация

### Настройки симуляции
```python

    'water_dynamics': {
        'flow_strength': 0.5,

        'turbulence': 0.1
    },
    'fish_behavior': {
```
