# Data Board (`data_board.py`)

Этот модуль реализует контроллер для визуализации отладочных данных в игровой среде с помощью OpenCV.

## Класс `DataBoard`

```python
class DataBoard(EntityController):
    def __init__(self):
        # ... реализация ...
```

### Наследование

Наследуется от `EntityController` (из `SLM.botbox.Environment`), что позволяет ему интегрироваться в игровой цикл и получать обновления.

### Зависимости

*   **OpenCV (`cv2`):**
    *   Для визуализации данных через окно OpenCV
    *   Отрисовка текста на изображении
*   **NumPy (`numpy`):**
    *   Создание буфера изображения
    *   Работа с массивами данных

### Атрибуты

*   `name`: Имя контроллера ('data_board')
*   `data`: Словарь для хранения отображаемых данных
*   `draw_image_buf`: Буфер изображения размером 640x480 пикселей
*   Частота обновления: 5 кадров в секунду (установлено через `SetUpdateRate(5)`)

### Методы

#### `__init__(self)`

```python
def __init__(self):
    super().__init__()
    self.name = 'data_board'
    self.data = {}
    self.SetUpdateRate(5)
    self.draw_image_buf = np.zeros((640, 480, 3), np.uint8)
```

**Действия:**
1. Инициализирует базовый класс
2. Устанавливает имя контроллера
3. Создает пустой словарь для данных
4. Устанавливает частоту обновления
5. Создает буфер изображения

#### `update(self)`

```python
def update(self):
    need_update = super().update()
    if not need_update:
        return
```

**Назначение:** 
- Проверяет необходимость обновления через базовый класс
- В текущей реализации не выполняет дополнительных действий

#### `render(self)`

```python
def render(self):
    return  # В текущей версии отключено
    self.draw_image_buf.fill(0)
    for i, (key, value) in enumerate(self.data.items()):
        cv2.putText(self.draw_image_buf, f"{key}: {value}", 
                    (10, i * 16 + 16), cv2.FONT_HERSHEY_SIMPLEX, 
                    0.5, (255, 255, 255), 1)
    cv2.imshow('DataBoard', self.draw_image_buf)
```

**Назначение:**
- Отрисовка данных в окне OpenCV (в текущей версии отключено)
- Каждая пара ключ-значение отображается на новой строке
- Очищает буфер перед каждой отрисовкой
- Использует простой шрифт и белый цвет для текста

### Использование

```python
# Создание доски данных
data_board = DataBoard()

# Добавление в игровую среду
environment.add_controller(data_board)

# Обновление данных
data_board.data["FPS"] = 60
data_board.data["Entities"] = 10
data_board.data["Memory"] = "125MB"
```

### Особенности реализации

1. **Интеграция с игровым циклом:**
   - Использует систему контроллеров из `Environment`
   - Поддерживает фиксированную частоту обновления

2. **Отображение:**
   - Использует OpenCV для визуализации
   - Построчный вывод пар ключ-значение
   - Поддерживает динамическое обновление данных

3. **Производительность:**
   - Буферизация изображения
   - Ограниченная частота обновления
   - Минимальное влияние на основной цикл

### Текущее состояние
