# Web Links (`web_link/`)

Этот подпакет отвечает за представление веб-ссылок (URI) как отдельных записей в базе данных SLM и предоставляет утилиты для извлечения информации с этих веб-страниц.

## Структура

*   [`__init__.py`](__init__.md): Пустой файл, необходимый для распознавания директории как пакета Python.
*   [`web_link_record.py`](web_link_record.md): Определяет классы:
    *   `WebLinkRecord`: Модель данных MongoDB (наследник `CollectionRecord`) для хранения информации о веб-ссылке (в основном, сам URI).
    *   `uriDecoder`: Класс с методами для извлечения данных с веб-страницы по URL (текст, скриншот, резюме с помощью LLM).
    *   `uriDecoderManager`: Управляет настройками и регистрацией декодеров URI.

## Назначение

Пакет `web_link` позволяет:
1.  **Хранить веб-ссылки:** Записывать и управлять URL как объектами `WebLinkRecord` в базе данных.
2.  **Извлекать контент:** Использовать `uriDecoder` для получения текстового содержимого, создания скриншотов и генерации кратких резюме веб-страниц.
3.  **Интегрировать веб-ресурсы:** Предоставляет основу для интеграции внешних веб-ресурсов в систему SLM, позволяя их каталогизировать, аннотировать и связывать с другими данными.

## Ключевые особенности

*   **Модель данных:** `WebLinkRecord` интегрируется с общей системой `CollectionRecord`.
*   **Извлечение данных:** Используются стандартные библиотеки (`requests`, `BeautifulSoup`) и внешние (`html2image`, `langchain`) для обработки веб-страниц.
*   **Суммаризация:** Применяется LLM (Mistral-7B) для автоматического создания резюме содержимого страницы.
*   **Инициализация:** Класс и коллекция регистрируются при запуске приложения через `allocator.Allocator`.
