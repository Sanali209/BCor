# Annotation Documentation (`Annotation/`)

Эта директория содержит существующие файлы документации в формате reStructuredText (`.rst`), описывающие структуру данных и рабочий процесс, связанные с аннотациями в `files_db`.

## Файлы

*   **`Annotation.rst`**: Описывает структуру записей для заданий аннотаций (`Annotation job`) и самих аннотаций (`Annotation record`), включая форматы ограничивающих рамок (`Box formats`).
*   **`anotation_types.rst`**: Предположительно, должен был содержать список типов аннотаций, но на данный момент содержит только заголовок.

**Примечание:** Эти файлы находятся в формате `.rst` и не были автоматически преобразованы в Markdown. Они предоставляют справочную информацию о структуре данных аннотаций в MongoDB.
