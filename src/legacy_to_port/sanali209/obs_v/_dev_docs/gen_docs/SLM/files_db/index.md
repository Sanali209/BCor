# Files Database (`files_db/`)

Пакет `files_db` отвечает за управление базой данных, содержащей информацию о файлах, их метаданных, результатах анализа (например, обнаруженные объекты, эмбеддинги) и связях между ними. Он тесно интегрирован с MongoDB через расширения `mongoext` и использует модульную систему `appGlue` для инициализации.

## Основные компоненты

*   [`files_db_module.py`](files_db_module.md): Определяет `FilesDBModule`, который инициализирует кэш эмбеддингов, регистрирует модели данных MongoDB (`FileRecord`, `Detection`, `TagRecord`, `RelationRecord` и др.), создает необходимые индексы и настраивает обработчики событий (например, для каскадного удаления).
*   [`__init__.py`](__init__.md): Регистрирует `FilesDBModule` в `Allocator` при импорте пакета, обеспечивая его своевременную загрузку и инициализацию.
