# Annotation Tool (`annotation_tool/`)

Этот пакет содержит инструменты и модели данных для управления процессом аннотирования (разметки) данных, в основном изображений, для задач машинного обучения. Он включает модели для хранения аннотаций в MongoDB, утилиты для экспорта аннотированных данных в различные форматы датасетов, а также скрипты для обучения и оптимизации гиперпараметров моделей классификации изображений на основе этих аннотаций.

## Основные модули Python

*   [`annotation.py`](annotation.md): Определяет основные классы `AnnotationJob` (задание аннотирования) и `AnnotationRecord` (отдельная аннотация), а также `SLMAnnotationClient` для управления заданиями и утилиты для работы с аннотациями.
*   [`annotation_export.py`](annotation_export.md): Содержит базовый класс `DataSetExporter`, менеджер `DataSetExporterManager` и конкретные реализации экспортеров для преобразования аннотаций в форматы датасетов (например, папки классов для классификации изображений).
*   [`singl_label_train.py`](singl_label_train.md): Скрипт для обучения (fine-tuning) модели классификации изображений (например, ViT) на аннотированных данных из `AnnotationJob` с использованием Hugging Face Transformers и PyTorch.
*   [`single_label_tune.py`](single_label_tune.md): Базовая версия скрипта для автоматического поиска гиперпараметров с использованием Ray Tune, без кэширования признаков.
*   [`singl_label_tune2.py`](singl_label_tune2.md): Улучшенная версия скрипта для поиска гиперпараметров с Ray Tune, включающая кэширование признаков с помощью `diskcache` для ускорения процесса.
*   [`__init__.py`](__init__.md): Пустой файл, необходимый для того, чтобы Python распознавал эту директорию как пакет.

## Поддиректории

*   `best_tuned_model/`: Вероятно, содержит артефакты (конфигурацию, веса) лучшей модели, найденной в результате работы скриптов оптимизации гиперпараметров (`*tune*.py`).
*   `predefained_jobs/`: Возможно, содержит предопределенные конфигурации или данные для `AnnotationJob`.
*   `training_output/`: Вероятно, используется скриптом `singl_label_train.py` для сохранения промежуточных или конечных результатов обучения (например, весов модели).

Этот пакет предоставляет комплексное решение для аннотирования данных внутри SLM и последующего использования этих аннотаций для обучения и оптимизации моделей машинного обучения.
