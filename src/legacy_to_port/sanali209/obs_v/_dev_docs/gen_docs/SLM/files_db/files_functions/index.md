# File Functions (`files_functions/`)

Этот подпакет содержит высокоуровневые функции, предназначенные для выполнения операций над файлами, зарегистрированными в базе данных SLM (`files_db`). Эти функции часто инкапсулируют более сложные рабочие процессы, такие как индексация папок или очистка базы данных.

## Структура

*   [`__init__.py`](__init__.md): Пустой файл, необходимый для распознавания директории как пакета Python.
*   [`delete_exist_file.py`](delete_exist_file.md): Содержит функцию `delete_existing` для удаления записей `FileRecord` из базы данных, если соответствующие им файлы больше не существуют в файловой системе. *(Планируется обернуть в `AppAction`)*.
*   [`index_folder.py`](index_folder.md): Содержит функции `index_folder_one_thread` и `index_folder` для запуска процесса индексации файлов (извлечение метаданных, генерация превью, эмбеддингов и т.д.) в указанной папке или для файлов, соответствующих определенному запросу. Поддерживает как последовательное, так и многопоточное выполнение.

## Назначение

Пакет `files_functions` предоставляет удобные точки входа для выполнения общих задач по управлению и обработке файлов в базе данных SLM:

1.  **Индексация:** Запуск процесса индексации для новых или существующих файлов, используя соответствующие "индексаторы" в зависимости от типа файла.
2.  **Очистка:** Удаление "осиротевших" записей из базы данных для поддержания ее актуальности.

Функции в этом пакете могут использоваться как утилиты командной строки (как показано в `index_folder.py`) или вызываться из других частей приложения, например, из пользовательского интерфейса или фоновых задач.
