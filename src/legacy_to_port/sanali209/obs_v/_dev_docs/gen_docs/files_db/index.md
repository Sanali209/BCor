# Документация по модулю `files_db`

Этот модуль отвечает за управление базой данных файлов, включая метаданные, индексацию, аннотации и связи.

## Основные разделы

*   [Архитектура](./architecture.md) - Обзор структуры и взаимодействия компонентов модуля.
*   [Основные концепции](./core_concepts.md) - Описание ключевых сущностей (File Record, Tag Record, Annotation Record) и структур данных.
*   [Компоненты](./components/index.md) - Детализация основных строительных блоков модуля (каталоги, связи, веб-ссылки).
*   [Индексаторы](./indexers/index.md) - Описание системы индексаторов для извлечения метаданных из файлов.
*   [Файловые операции](./file_operations.md) - Функции для работы с файлами и папками (индексация, удаление).
*   [Инструмент аннотации](./annotation_tool.md) - Описание возможностей по аннотированию данных.
*   [Распознавание объектов](./object_recognition.md) - Функционал для детекции объектов на изображениях.
*   [Интеграция с векторной БД](./vector_db.md) - Взаимодействие с векторной базой данных.
*   [Главный модуль (`files_db_module.py`)](./main_module.md) - Описание основного интерфейса модуля (если применимо).
*   [Рекомендации по развитию](./future_development.md) - Предложения по улучшению и доработке модуля.
